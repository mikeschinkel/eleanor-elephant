Meet Eleanor Elephant.

Fast, simple and super powerful Vagrant local environment for WordPress.

Linux, Windows, Mac

The most simple vagrant box you have ever worked with.

Experience the difference with Eleanor Elephant features and let fall in love with JungleWP 's local pod:
PHP 7.2, Ubuntu-17.10!, APACHE: 2.4.29, NGINX: 1.13.8, RUBY via RVM: 2.5.0, NODE via NVM: 8.9.4

Easy to install

Eleanor Elephant is a pre-configured Vagrant Box with a full array of tools to get you started with WordPress development.

Bloat Free

Eleanor Elephant is designed to be minimal, yet powerful, with bleeding-edge features tailored for Web technologies.

You are in Control

Use your favorite terminal console to run any command on your Pod just like on remote servers (using SSH)

Start developing in no time

This box should have all your needs for doing most development so you don't have to worry about configuring Vagrant and you can simply focus on writing code.

Simple and Reliable

No provisioning tools or setup is required. Everything is packaged into the box, running "vagrant" is super fast, and you won't need Internet to code.
